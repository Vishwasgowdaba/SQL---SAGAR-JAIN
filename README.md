# Student Database Normalization (1NF, 2NF, 3NF)

## Project Overview

This project demonstrates the process of **database normalization** using a student dataset.
The dataset is normalized step-by-step from an unnormalized structure to **First Normal Form (1NF)**, **Second Normal Form (2NF)**, and **Third Normal Form (3NF)**.

The goal is to reduce **data redundancy**, improve **data integrity**, and organize the database efficiently.

---

## Concepts Covered

* First Normal Form (1NF)
* Second Normal Form (2NF)
* Third Normal Form (3NF)
* Primary Keys
* Composite Keys
* Foreign Keys
* SQL Table Creation
* SQL Insert Queries

---

## Learning Outcome

By completing this project, you will understand:

* How to normalize a database
* How to design relational tables
* How to reduce redundancy in databases
* How to implement normalized schemas using SQL
